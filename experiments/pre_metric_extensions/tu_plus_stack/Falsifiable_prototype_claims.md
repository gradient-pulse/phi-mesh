# Falsifiable Prototype Claims

1. A TU-only loop can map trains and couplings, but will be weak at significance discrimination.

2. Adding TU+ should improve:
- likely continuation detection
- novelty vs familiarity discrimination
- instability flagging
- recruitment of cortexLLM attention

3. A cortexLLM reading TU+ signals should interpret unfolding scenes more coherently than a cortexLLM reading raw frame summaries alone.

4. Prompt-instantiated TU / TU+ / cortexLLM roles can remain distinct if:
- state is persistent
- outputs are schema-bounded
- recursion is looped but role-bounded

5. If the roles collapse into generic prose assistants, the architecture has failed its first practical test.
