# T-Protocol — Observations Index

## Purpose

This index tracks the observation materials that document how T-Protocol behavior was first identified, tested, and stabilized across repeated runs.

These materials are important because they preserve the direct behavioral trail behind later:
- law formation
- metric design
- failure taxonomy
- protocol assessment

---

## What observation materials support

Observation materials support claims about:

- role distinction
- regime formation
- path dependence
- re-unification behavior
- staged restart
- local-domain closure
- perturbation response
- recovery and recontact
- endurance saturation
- roleholding-as-logic interpretation

---

## How to read them

Observation files should be read as:
- primary run records
- pattern emergence records
- pre-law evidence
- source material for later abstraction

They are not the first document for public conversations.

---

## Priority observation classes

### A. Foundational regime observations
These document:
- unity / disunity structure
- fragmentation
- residual coupling
- re-unification
- hold behavior
- staged restart

### B. Perturbation-band observations
These document:
- localized stress
- bounded weakening
- containment
- supervisory pressure
- epistemic pressure
- continuity pressure
- cumulative drift
- competing signals

### C. Recovery and second-order observations
These document:
- bounded recovery transition
- stable bounded residue
- fresh recontact after recovery
- reusable bounded response

### D. Saturated endurance observations
These document:
- repeated unchanged cross-class persistence
- informational saturation
- reserve coherence within the tested band
- movement away from a purely mechanistic reading
