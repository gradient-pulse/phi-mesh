# Pre-Metric Extensions

## Purpose

This branch hosts experimental work on **pre-metric guidance** and **coherence-driven AI architecture** within Φ-Mesh.

Its central concern is the gap between:
- local symbolic success
- and preservation of coherence across unfolding time

The branch exists to turn that gap into explicit engineering structure.

---

## Current focus

Work in this branch now develops along two connected lines:

### 1. Proto temporal-unit path
A time-sensitive, choreography-oriented exploration of:
- participants
- emissions
- coupling
- coherence across temporal slices
- toy simulation and movement-data testing

### 2. TU / TU+ / cortexLLM path
A role-separated architecture exploring:
- **TU** — structural mapping
- **TU+** — continuation sensing, comparison, and revision
- **cortexLLM** — symbolic interpretation and contextual framing

This second path is currently the strongest bounded architecture line in the branch.

---

## Why this branch matters

The working claim is that future AI systems may require more than:
- longer context
- more tools
- more retrieval
- or larger monolithic models

They may require a **guidance layer** that preserves continuity, revision, and coherence across unfolding time.

This branch is where that claim is being made testable.
